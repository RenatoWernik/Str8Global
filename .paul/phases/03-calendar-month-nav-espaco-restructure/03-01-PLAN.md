    O problema: Atualmente o calendário navega apenas dia-a-dia com handlePrev/handleNext (+/-1 dia).
    Para ir ao próximo mês, o utilizador precisa clicar ~30 vezes no botão de próximo dia.

    Correção — Adicionar botões de mês à barra de navegação:

    1. Adicionar duas funções de navegação por mês (junto das funções handlePrev/handleNext existentes):

       ```typescript
       const handlePrevMonth = () => {
         const current = new Date(selectedDate + 'T00:00:00');
         const firstOfPrevMonth = new Date(current.getFullYear(), current.getMonth() - 1, 1);
         const todayDate = new Date(today + 'T00:00:00');
         // Se o 1º do mês anterior é antes de hoje, vai para hoje
         if (firstOfPrevMonth < todayDate) {
           setSelectedDate(today);
         } else {
           setSelectedDate(firstOfPrevMonth.toISOString().split('T')[0]);
         }
       };

       const handleNextMonth = () => {
         const current = new Date(selectedDate + 'T00:00:00');
         const firstOfNextMonth = new Date(current.getFullYear(), current.getMonth() + 1, 1);
         const maxDateObj = new Date(maxDate + 'T00:00:00');
         // Se o 1º do próximo mês excede maxDate, vai para maxDate
         if (firstOfNextMonth > maxDateObj) {
           setSelectedDate(maxDate);
         } else {
           setSelectedDate(firstOfNextMonth.toISOString().split('T')[0]);
         }
       };
       ```

    2. Adicionar variáveis de controlo para desativar botões de mês:
       ```typescript
       const currentMonth = new Date(selectedDate + 'T00:00:00').getMonth();
       const todayMonth = new Date(today + 'T00:00:00').getMonth();
       const todayYear = new Date(today + 'T00:00:00').getFullYear();
       const currentYear = new Date(selectedDate + 'T00:00:00').getFullYear();
       const canGoPrevMonth = !(currentMonth === todayMonth && currentYear === todayYear);
       const maxDateObj = new Date(maxDate + 'T00:00:00');
       const canGoNextMonth = new Date(currentYear, currentMonth + 1, 1) <= maxDateObj;
       ```

    3. Na barra de navegação de datas (div com flex items-center gap-2, linha ~174),
       adicionar uma segunda linha abaixo com navegação por mês. APÓS o div de navegação diária
       e ANTES da secção de atalhos/legenda, adicionar:

       ```tsx
       {/* Month navigation - desktop only */}
       <div className="hidden md:flex items-center justify-center gap-3 w-full md:w-auto">
         <button
           onClick={handlePrevMonth}
           disabled={!canGoPrevMonth}
           className={`px-3 py-1.5 rounded-lg border text-xs font-medium transition-all ${
             canGoPrevMonth
               ? 'bg-white/[0.05] hover:bg-white/[0.1] border-white/10 hover:border-white/20 text-white/70 hover:text-white active:scale-95'
               : 'bg-transparent border-transparent text-white/20 cursor-not-allowed'
           }`}
           aria-label="Mês anterior"
         >
           ← Mês anterior
         </button>
         <span className="text-xs text-white/40 font-medium uppercase tracking-wider">
           {MONTHS_PT[currentMonth]} {currentYear}
         </span>
         <button
           onClick={handleNextMonth}
           disabled={!canGoNextMonth}
           className={`px-3 py-1.5 rounded-lg border text-xs font-medium transition-all ${
             canGoNextMonth
               ? 'bg-white/[0.05] hover:bg-white/[0.1] border-white/10 hover:border-white/20 text-white/70 hover:text-white active:scale-95'
               : 'bg-transparent border-transparent text-white/20 cursor-not-allowed'
           }`}
           aria-label="Próximo mês"
         >
           Próximo mês →
         </button>
       </div>
       ```

    A navegação por mês aparece APENAS em desktop (`hidden md:flex`).
    A navegação por dia (chevrons) mantém-se inalterada em ambas as versões.

    NÃO alterar a lógica de disponibilidade ou slots.
    NÃO alterar o MobileBottomSheet (plano 02-01 já trata mobile).

       ```tsx
       'use client';

       import { useState } from 'react';
       import { motion } from 'framer-motion';
       import { RevealImage } from '@/components/animations/RevealImage';
       import { TrueFocus } from '@/components/animations/TrueFocus';
       import { BalancedHeadline } from '@/components/animations/BalancedHeadline';
       import { ImageLightbox } from '@/components/ui/ImageLightbox';
       import { MagneticCursor } from '@/components/ui/MagneticCursor';
       import { Masonry } from '@/components/ui/Masonry';
       import { useSectionInView } from '@/hooks/useSectionInView';
       import { podcastImages, espacoContent } from '@/data/espacoData';

       export function PodcastSection() {
         const { sectionRef, isInView } = useSectionInView();
         const [selectedImage, setSelectedImage] = useState<number | null>(null);

         return (
           <section ref={sectionRef} className="relative py-24 px-6 bg-[#030303] z-20 border-t border-white/5">
             <div className="max-w-7xl mx-auto">
               <div className="mb-16 flex flex-col items-end text-right">
                 <TrueFocus
                   sentence={espacoContent.podcast.title}
                   manualMode={false}
                   blurAmount={5}
                   borderColor="#22d3ee"
                   glowColor="rgba(34, 211, 238, 0.4)"
                   animationDuration={0.8}
                   pauseBetweenAnimations={1.5}
                   className="text-3xl md:text-5xl font-bold mb-4 justify-end"
                 />
                 <BalancedHeadline as="p" className="text-white/60 text-lg max-w-2xl ml-auto">
                   {espacoContent.podcast.subtitle}
                 </BalancedHeadline>
               </div>

               {/* Desktop: 3 columns */}
               <div className="hidden md:block">
                 <Masonry columns={3} gap={20}>
                   {podcastImages.map((src, idx) => (
                     <MagneticCursor key={idx} strength={0.15} className="w-full !block mb-5">
                       <motion.div
                         layoutId={`podcast-${idx}`}
                         className="relative overflow-hidden rounded-xl border border-white/5 cursor-pointer w-full"
                         onClick={() => setSelectedImage(idx)}
                       >
                         <RevealImage
                           src={src}
                           alt={`Estúdio de Podcast ${idx + 1}`}
                           isInView={isInView}
                           delay={idx * 0.15}
                           direction="up"
                           useFill={false}
                           width={1000}
                           height={1000}
                           imageClassName="transition-all duration-700 hover:scale-105"
                         />
                       </motion.div>
                     </MagneticCursor>
                   ))}
                 </Masonry>
               </div>

               {/* Mobile: 1 column */}
               <div className="md:hidden block">
                 <Masonry columns={1} gap={16}>
                   {podcastImages.map((src, idx) => (
                     <div
                       key={idx}
                       className="relative overflow-hidden rounded-xl border border-white/5 cursor-pointer w-full"
                       onClick={() => setSelectedImage(idx)}
                     >
                       <RevealImage
                         src={src}
                         alt={`Estúdio de Podcast ${idx + 1}`}
                         isInView={isInView}
                         delay={idx * 0.1}
                         direction="up"
                         useFill={false}
                         width={800}
                         height={800}
                       />
                     </div>
                   ))}
                 </Masonry>
               </div>
             </div>

             {selectedImage !== null && (
               <ImageLightbox
                 src={podcastImages[selectedImage]}
                 alt={`Estúdio de Podcast ${selectedImage + 1}`}
                 layoutId={`podcast-${selectedImage}`}
                 isOpen={selectedImage !== null}
                 onClose={() => setSelectedImage(null)}
               />
             )}
           </section>
         );
       }
       ```

       Nota: A secção Podcast alinha título/subtítulo à DIREITA (flex items-end text-right,
