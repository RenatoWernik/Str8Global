**Layout — Staggered Alternating Offset (5 images):**

A zigzag/offset layout where images alternate between wide-left and narrow-right, creating visual rhythm completely different from the bento grid:

```
Desktop layout (md+):
┌────────────────────┐
│ img1 (75% width)   │  ← aligned left
│                    │
└────────────────────┘
          ┌────────────────────┐
          │ img2 (60% width)   │  ← aligned right
          └────────────────────┘
┌──────────────────────────┐
│ img3 (80% width, taller) │  ← aligned left
│                          │
└──────────────────────────┘
          ┌──────────────┐
          │ img4 (50%)   │  ← aligned right
          └──────────────┘
┌────────────────────────────────┐
│ img5 (full width, shorter)     │  ← full bleed
└────────────────────────────────┘
```

Implementation using flexbox with alternating alignment:
- Container: `flex flex-col gap-6 md:gap-8`
- Each image row: a div with flex properties controlling alignment
- Odd images (0, 2, 4): `self-start` (left-aligned)
- Even images (1, 3): `self-end` (right-aligned)

**Width and aspect ratio per image (desktop):**
- Image 1 (CoworkGeral): `md:w-[75%]`, `aspect-[16/10]` — wide landscape, left
- Image 2 (CoworkGeral2): `md:w-[60%]`, `aspect-[4/3]` — medium, right
- Image 3 (Cowork-Starter): `md:w-[80%]`, `aspect-[16/9]` — wide landscape, left
- Image 4 (Cowork-Prime): `md:w-[50%]`, `aspect-[3/2]` — narrower, right
- Image 5 (Cowork-Premium): `md:w-full`, `aspect-[21/9]` — full-width cinematic, center

**Mobile:** All images `w-full` with `aspect-video`.
