**Layout — Full-Bleed Horizontal Showcase (3 images):**

Three large images side by side (desktop) creating an immersive gallery feel. Different from bento (asymmetric grid) and staggered (zigzag offset). The focus is on large-format images that feel premium.

```
Desktop (md+):
┌──────────────┬──────────────┬──────────────┐
│              │              │              │
│   img1       │   img2       │   img3       │
│  (tall)      │  (tall)      │  (tall)      │
│              │              │              │
│  hover info  │  hover info  │  hover info  │
└──────────────┴──────────────┴──────────────┘
```

But NOT a simple equal 3-column grid. Use varying proportions:
- Container: `grid grid-cols-1 md:grid-cols-10 gap-3 md:gap-4`
- Image 1 (Sol): `md:col-span-4` — slightly wider
- Image 2 (Cafe): `md:col-span-3` — narrower (creates asymmetry)
- Image 3 (Maquiagem): `md:col-span-3` — same as img2

All images use `aspect-[3/4]` on desktop (tall portrait format) — this creates tall immersive cards, visually distinct from the landscape formats in Studios and Cowork.

**Each card structure:**
```tsx
<div className="group relative overflow-hidden rounded-2xl border border-white/10">
  <RevealImage
    src={img.src}
    alt={img.title}
    isInView={isInView}
    delay={idx * 0.15}
    direction={idx === 0 ? 'left' : idx === 1 ? 'up' : 'right'}
    className="relative w-full aspect-[3/4]"
    imageClassName="transition-all duration-700 opacity-80 group-hover:opacity-100 group-hover:scale-105"
  />
  <div className="absolute inset-0 bg-gradient-to-t from-black/80 via-black/20 to-transparent" />
  <div className="absolute bottom-0 left-0 p-6 md:p-8 w-full">
    <h3 className="text-2xl md:text-3xl font-semibold mb-2 text-white">{img.title}</h3>
    <div className="w-12 h-1 bg-[var(--color-accent)] transform origin-left transition-transform duration-300 scale-x-0 group-hover:scale-x-100 mb-3" />
    <p className="text-white/70 text-sm md:text-base leading-relaxed opacity-0 transform translate-y-4 transition-all duration-300 group-hover:opacity-100 group-hover:translate-y-0">
      {img.desc}
    </p>
  </div>
</div>
```
