    ```typescript
    export async function getMonthlyAvailability(params: {
      item_id?: string;
      plan_id?: string;
      month: string; // "YYYY-MM"
    }): Promise<string[]>
    ```

    Logic:
    - Parse month to get `firstDay` (YYYY-MM-01) and `lastDay` (YYYY-MM-28/29/30/31)
    - If `item_id` provided: query `reservations` where `item_id = params.item_id`, `status = 'active'`, date range overlaps the month (`start_date <= lastDay AND end_date >= firstDay`)
      - For each reservation, compute all dates within the month that fall in [start_date, end_date] range
      - For studio reservations with `start_time` (hourly), count how many hours are booked per day; mark day as unavailable only if ALL hours (8h-23h = 15 slots) are booked
      - For gear reservations (no time), every date in the range is unavailable
      - Return deduplicated sorted array of date strings
    - If `plan_id` provided: query `cowork_reservations` where `plan_id = params.plan_id`, `status = 'active'`, date range overlaps month
      - Also fetch capacity for that plan via `getCoworkCapacity()`
      - For each date in the month, sum up `spots` from active reservations covering that date
      - If sum >= totalSpots → that date is unavailable
      - Return array of unavailable date strings
    - If neither `item_id` nor `plan_id` → throw error

    ```typescript
    export async function getHourlyAvailability(
      studioId: string,
      date: string // "YYYY-MM-DD"
    ): Promise<Array<{ hour: string; available: boolean; reservation?: string }>>
    ```

    Logic:
    - Define hours array: ["08:00", "09:00", ..., "22:00"] (15 one-hour slots, 8h-23h)
    - Query `reservations` where `item_id = studioId`, `status = 'active'`, `start_date <= date`, `end_date >= date`
    - For each hour slot, check if any reservation covers it:
      - A reservation covers hour H if: `start_time <= H` AND `end_time > H`
      - For day-based studio reservations (no start_time/end_time), ALL hours are occupied
    - Return array of slot objects: `{ hour: "10:00", available: true/false, reservation: "Client Name" }`
    - The `reservation` field contains `client` name from the reservation (only when occupied)

    **2. Create `src/app/api/rental/availability/hourly/route.ts`:**

    - `GET` handler accepting query params: `studio_id` (required), `date` (required, YYYY-MM-DD)
    - Validate both params, return 400 if missing/invalid
    - Call `getHourlyAvailability(studioId, date)`
    - Return: `{ date, studio_id, slots: [{ hour: "08:00", available: true }, { hour: "10:00", available: false, reservation: "Cliente X" }, ...] }`
    - Cache: `Cache-Control: public, s-maxage=300, stale-while-revalidate=60`
    - Error fallback: return all slots as available `{ available: true }` on error (graceful degradation)
    - Export `revalidate = 300`
